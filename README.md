# Date-Tracker-Gift-and-Date-Tracking-System
This project is a gift and date tracking system designed to help users manage birthdays, holidays, and special occasions for large groups of people in one central location. By combining a digital calendar with gift-idea logs for each individual, this eliminates last-minute rushing and helps users stay organized.

## Features
- Centralized Profiles: Manage friends/family details in one location.
- Date Tracking: Auto-log birthdays, anniversaries, and holidays.
- Advanced Alerts: Receive notifications early to avoid last-minute rush.
- Gift Log: Instantly save gift ideas to specific profiles.
- Summary Dashboard: View all upcoming dates and gifts immediately.
- Status Tracker: Mark items as "purchased" to stay organized.
